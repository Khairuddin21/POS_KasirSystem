FLOW:
1. Pilih kategori (opsional)
2. Cari produk (opsional) 
3. Klik produk → Masuk keranjang
4. Atur quantity [+] [-]
5. Pilih metode pembayaran
6. Input uang bayar
7. Klik "Proses Pembayaran"
8. Modal sukses muncul
9. Print struk (opsional)
10. Transaksi baru

KEYBOARD SHORTCUTS:
- F2: Focus Search
- F3: Focus Payment Input
- F4: Process Transaction

🔄 API ENDPOINTS (KASIR):
═══════════════════════════════════════════════════════════════════════════════

POST /kasir/transaction/process
├── Input:
│   ├── items[] (product_id, quantity)
│   ├── payment_method (cash/card/qris/transfer)
│   └── paid (amount)
└── Output:
    ├── success (boolean)
    ├── message (string)
    └── transaction (object)
        ├── transaction_code
        ├── total
        ├── paid
        ├── change
        └── items[]
